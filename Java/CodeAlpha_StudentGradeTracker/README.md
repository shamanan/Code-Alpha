# CodeAlpha - Student Grade Tracker

A simple Java console application to track and manage student grades.

## 📁 Project Structure

```
CodeAlpha_StudentGradeTracker/
├── src/
│   └── Main.java
└── README.md
```

## 🚀 Features

- Add student names and their grades
- View all students with letter grades (A, B, C, D, F)
- Calculate average grade
- Find highest and lowest grade

## 🛠️ How to Run

### Using Command Prompt / Terminal

1. Navigate to the `src` folder:
   ```
   cd path\to\CodeAlpha_StudentGradeTracker\src
   ```

2. Compile the Java file:
   ```
   javac Main.java
   ```

3. Run the program:
   ```
   java Main
   ```
